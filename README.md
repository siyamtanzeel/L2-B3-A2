# assignment-2

To install dependencies:

```bash
npm install
```

To run in development mode:

```bash
npm run dev
```

To run lint for finding errors in the code:

```bash
npm run lint
```

To fix errors in the code:

```bash
npm run fix
```

To build the project:

```bash
npm run build
```

To run production build:

```bash
npm run start
```
